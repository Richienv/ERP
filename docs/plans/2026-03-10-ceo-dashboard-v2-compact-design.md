# CEO Dashboard v2 — Compact Dense Grid Design

**Goal:** Replace monotonous 6-strip layout with a compact dense grid: KPI pills at top, 2×3 module cards with mixed heights, inline PO approval, recent order lists, and tasks integrated into the grid.

**Layout:** Greeting + 5 KPI pills → 2×3 module grid (Penjualan, Pengadaan, Inventori, Manufaktur, SDM, Tugas) → No separate activity feed.

**Key Features:**
- Finance becomes compact KPI pills (not a full strip)
- Pengadaan card has inline PO Setujui/Tolak buttons
- Penjualan card shows 2-3 recent orders
- Tugas Hari Ini moves into the 6th grid slot
- No Activity Feed (tasks already cover what needs attention)
- Mixed card heights create natural visual variety
